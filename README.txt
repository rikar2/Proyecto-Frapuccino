1. INTRODUCCIÓN

Frapuccino es un programa para Smartphone, especificamente BlackBerry
que permite ubicar lugares geográficos tales como: avenidas, calles, ciudades, etc.
Realizar trazabilidades entres lugares, ver la ubicación del usuario mediante el uso del
GPS con los que cuentan estos dispositivos. Por otra parte integra busqueda de lugares comerciales
dentro de un país, ciudad, avenida, etc. 

Frapuccino integra dos sistemas, uno de ubicación Geográfica y otro de ubicación comercial.

El programa esta desarrollado en Jave ME con algunas api de Research in Motion.


2. INSTALACIÓN

Bajar el archivo Frapuccino.cod y Frapuccino.alx http://github.com/rikar2/Proyecto-Frapuccino/tree/master/bin/java%20ME/
en el mismo directorio los dos archivos, con el Desktop Manager de Blackberry, seleccionan añadir aplicación, se busca
el archivo Frapuccino.alx y continuan.
